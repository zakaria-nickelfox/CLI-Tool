Problem: "geninit: command not found"
Solution: 
  1. Make sure virtual environment is activated
  2. Run: pip install -e .
  3. Try: python -m geninit.main

Problem: "No module named 'typer'"
Solution: 
  1. Run: pip install -r requirements.txt
  2. Then: pip install -e .

Problem: "scripts: command not found" (PowerShell)
Solution: 
  1. Use: .\scripts install (with the dot-slash)
  2. Or use CMD: scripts install
  3. Or run directly: pip install -e .

Problem: "scripts: command not found" (CMD)
Solution: 
  1. Use: scripts.bat install
  2. Or run directly: pip install -e .

Problem: "make: command not found" (Windows)
Solution: 
  1. Use scripts.bat instead: scripts install
  2. Or install make for Windows
  3. Or run commands directly
